# AI_YouTube_MultiAgent_Project

A simple Python-based multi-agent system for generating YouTube content using sample data.
Built for the Kaggle Hackathon Capstone.

# Files

HACKTHON CAPSTONE PROJECT PROG.ipynb — Main notebook
sample_youtube_dataset.csv — Sample data
project_summary.docx — Summary document

# Run the Notebook
Run on Kaggle:
https://www.kaggle.com/code/ratanagarwal/hackathon-capstone-project

Or run locally after installing:
pip install pandas numpy matplotlib python-docx
